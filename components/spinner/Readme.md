## Basic use

```vue
<template>
  <div>
    <yo-spinner type="Android"></yo-spinner>
    <yo-spinner type="iOS"></yo-spinner>
    <yo-spinner type="Windows"></yo-spinner>
  </div>
</template>
```

## Size

```vue
<template>
  <div>
    <yo-spinner type="Android" size="32px"></yo-spinner>
    <yo-spinner type="iOS" size="32px"></yo-spinner>
    <span style="font-size: 32px;">
      <yo-spinner type="Windows"></yo-spinner>
    </span>
  </div>
</template>
```

## Color

```vue
<template>
  <div style="font-size: 32px">
    <yo-spinner type="Android" color="#1b98e0"></yo-spinner>
    <yo-spinner type="iOS" color="#ff7f00"></yo-spinner>
    <span style="color: #37af79">
      <yo-spinner type="Windows"></yo-spinner>
    </span>
  </div>
</template>
```
